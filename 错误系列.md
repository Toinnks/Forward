### 服务器错误系列：

#### YOLO环境配置，openCV包冲突：

AttributeError: partially initialized module 'cv2' has no attribute 'gapi_wip_gst_GStreamerPipeline'
(most likely due to a circular import)

**原因:**

系统里同时安装了多个版本或冲突的 `opencv` 相关包（比如 `opencv-python`、`opencv-contrib-python`、`opencv-python-headless`），导致 **OpenCV 模块部分加载失败**，`cv2` 内部引用错乱。

**解决措施：**

1️⃣ 彻底卸载所有 OpenCV 残留包（确保无冲突）

pip uninstall -y opencv-python opencv-python-headless opencv-contrib-python opencv-contrib-python-headless

2️⃣ 清理可能的 cv2 目录

rm -rf /usr/local/lib/python3.8/dist-packages/cv2*
rm -rf /usr/local/lib/python3.8/site-packages/cv2*
rm -rf /usr/lib/python3.8/site-packages/cv2*

3️⃣ 重新安装兼容版本（YOLOv8 推荐）

pip install --no-cache-dir opencv-python-headless==4.8.1.78 -i https://pypi.tuna.tsinghua.edu.cn/simple

4️⃣ 验证

python3 -c "import cv2; print(cv2.__file__, cv2.__version__)"

正常会输出:/usr/local/lib/python3.8/dist-packages/cv2/python-3.8/cv2.cpython-38-x86_64-linux-gnu.so 4.8.1
